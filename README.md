# 🔥 Auto - Facebook Account Creator Tool

> **Simple Python Tool To Create FB IDs**

## 📌 Overview

Auto is a powerful Facebook account automation tool designed for creating Facebook accounts efficiently. Built with Python and optimized with Cython for maximum performance.

## ✨ Features

- ✅ **Facebook Account Creation** - Create multiple Facebook accounts automatically
- ✅ **Auto Friend Request** - Send bulk friend requests using cookies
- ✅ **Hotmail/Outlook Checker** - Check Outlook/Hotmail messages via Microsoft Graph API
- ✅ **TXT to XLSX Converter** - Convert account data to Excel format with append mode
- ✅ **Mixed Domain Email Generator** - Generate emails with multiple domains
- ✅ **Nepali Names Support** - Realistic Nepali name generation
- ✅ **Cython Optimized** - Compiled for faster execution
- ✅ **Round Robin System** - Efficient friend request sending

## 📋 Requirements

- Python 3.13 or higher
- Termux (Android) or Linux terminal
